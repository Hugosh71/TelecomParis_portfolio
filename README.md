# Hugo Fanchini — Portfolio

Personal portfolio built with [Remix](https://remix.run/), [Three.js](https://threejs.org/), and [Framer Motion](https://www.framer.com/motion/).

## Install & run

Requires nodejs `19.9.0` or higher and npm `9.6.3` or higher.

```bash
npm install
npm run dev
```

## Deployment

Deployed on Cloudflare Pages:

```bash
npm run deploy
```
